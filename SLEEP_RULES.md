# Sleep rules

## Important note

This app is not a medical tool. It provides configurable planning assistance based on parent-defined targets.

## Default preset: 6 months

Target wake time: 07:00  
Target bedtime: 19:30  
Target total awake time: 630 minutes  
Allowed total awake time: 600-660 minutes  
Target day sleep: 180 minutes  
Allowed day sleep: 150-240 minutes  
Default nap count: 3  
Micro-nap duration: 15-30 minutes  
Earliest bedtime: 18:30  
Latest bedtime: 20:30  

## Wake window defaults

Wake window 1: 120-150 minutes  
Wake window 2: 135-165 minutes  
Wake window 3: 150-180 minutes  
Last wake window: 165-210 minutes  

## Short nap

A nap under 40 minutes is considered short.

## Micro-nap

A micro-nap is a short nap of 15-30 minutes used to bridge the gap to bedtime.

Use a micro-nap scenario when:
- bedtime would otherwise be too early;
- the last wake window would become too long;
- total awake time would exceed the configured maximum;
- there is still enough time before bedtime.

## Early bedtime

Use early bedtime scenario when:
- it is too late for a micro-nap;
- day sleep is low;
- wake time is already high;
- normal bedtime would create an excessive last wake window.

## Recommendation style

Recommendations should be practical and short.

Bad:
"Your child's circadian rhythm requires recalibration."

Good:
"Первый сон был короткий. Лучше попробовать следующий сон чуть раньше и держать в запасе микросон."
